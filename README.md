### 项目启动

基础的 **HTML/CSS/JS** 页面，可以直接浏览器打开，也可以使用 live-server 启动。

全局安装：

```bash
npm i live-server -g
```

当前目录启动（index.html为入口）：

```bash
live-server
```

### 项目压缩

安装开发依赖：

```bash
npm i
```

压缩代码：

```bash
npx gulp
```
